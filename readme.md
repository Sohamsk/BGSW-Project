# VB6 to C# converter
```A code transpiler that transpiles VB6 to C#```

Requirements
 - go 1.23

To use:
1. Run ```make setup``` to setup the project.
2. Run ```make build``` to build the project.
3. Run ```make test``` to run a sanity test.
4. Run ```make clean``` to clean.

- Windows users can use the binary provided in the releases tab or build one using the  
commands in the build target of the make file.

To run a proper test run ```auto_test_script.py```
